# Tetris_project 

## C언어를 이용하여 테트리스 만들기 
겨울 프로젝트 > Debug > 겨울 프로젝트.exe


## 1. ABOUT 
##### * NAME: TETRIS 
##### * GENRE: PUZZLE 
##### * PLATFORM: PC(Consule) 
##### * LANGUAGE: C(Visual Studio) 

## 2. FEATURE 
##### * 수평으로 한 줄 채우면 점수를 얻는 방식 
##### * Hard drop를 이용하여 블록을 채우면 보너스 점수 
##### * 일정 점수를 획득하면 다음 단계로 넘어감 
##### * 단계가 올라갈수록 블록 내려오는 속도가 빨라짐
